class xspline.bspl.Bspl(params)[source]#

Bases: BundleXFunction

Basis spline function.

Parameters:

params (tuple[tuple[float, ...], int, int]) – This is a tuple that contains knots, degree and index of the basis function.

Example

>>> bspl = Bspl(((0.0, 1.0), 1, 0)) # knots=(0.0, 1.0), degree=2, index=0
>>> bspl([0.0, 1.0])
array([0., 1.])
>>> bspl([0.0, 1.0], order=1)
array([1., 1.])
>>> bspl([0.0, 1.0], order=2)
array([0., 0.])
>>> bspl([0.0, 1.0], order=-1)
array([0. , 0.5])
